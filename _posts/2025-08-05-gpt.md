## 1. Controller — 요청 수신 및 인증 검사 (Spring Security)

```java
@RestController
@RequestMapping("/api/secrets")
public class SecretController {

    private final SecretService secretService;
    private final IpExtractor ipExtractor; // 클라이언트 IP 추출 유틸

    public SecretController(SecretService secretService, IpExtractor ipExtractor) {
        this.secretService = secretService;
        this.ipExtractor = ipExtractor;
    }

    @GetMapping
    public ResponseEntity<?> getSecret(
        @RequestParam String account,
        @RequestParam String secretPath,
        HttpServletRequest request
    ) {
        String clientIp = ipExtractor.extract(request);

        try {
            String secret = secretService.getSecret(account, secretPath, clientIp);
            return ResponseEntity.ok(Map.of("secret", secret));
        } catch (AccessDeniedException e) {
            return ResponseEntity.status(HttpStatus.FORBIDDEN).body("IP not allowed");
        } catch (SecretNotFoundException e) {
            return ResponseEntity.status(HttpStatus.NOT_FOUND).body("Secret not found");
        } catch (Exception e) {
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).body("Internal error");
        }
    }
}
```

## 2. Service — 비즈니스 로직: IP검증, 시크릿 조회

```java
@Service
public class SecretService {

    private final OpenBaoMetadataClient openBaoMetadataClient;
    private final ExtensionClient extensionClient;

    public SecretService(OpenBaoMetadataClient openBaoMetadataClient, ExtensionClient extensionClient) {
        this.openBaoMetadataClient = openBaoMetadataClient;
        this.extensionClient = extensionClient;
    }

    public String getSecret(String account, String secretPath, String clientIp) {
        String namespace = mapAccountToNamespace(account);

        // 1. OpenBao 메타데이터에서 허용 IP 리스트 조회
        List<String> allowedIps = openBaoMetadataClient.getAllowedIps(namespace, "manager/iprestriction");

        // 2. IP 허용 여부 검사
        if (!allowedIps.contains(clientIp)) {
            throw new AccessDeniedException("IP not allowed: " + clientIp);
        }

        // 3. 허용된 IP면 Extension API 호출하여 시크릿 조회
        String secret = extensionClient.fetchSecret(account, secretPath);

        if (secret == null || secret.isEmpty()) {
            throw new SecretNotFoundException("Secret not found for path: " + secretPath);
        }

        return secret;
    }

    private String mapAccountToNamespace(String account) {
        // account → 네임스페이스 매핑 로직 (필요시 DB 또는 config)
        return account + "-ns";
    }
}
```

## 3. OpenBaoMetadataClient — IP 리스트 조회 (Vault/KV v2 메타데이터 호출)

```java
@Component
public class OpenBaoMetadataClient {

    private final RestTemplate restTemplate;

    public OpenBaoMetadataClient(RestTemplate restTemplate) {
        this.restTemplate = restTemplate;
    }

    public List<String> getAllowedIps(String namespace, String ipRestrictionPath) {
        String url = String.format("https://openbao.example.com/v1/%s/kv-v2/metadata/%s", namespace, ipRestrictionPath);

        HttpHeaders headers = new HttpHeaders();
        headers.set("X-Vault-Namespace", namespace);
        headers.set("X-Vault-Token", "<vault_token>");

        HttpEntity<Void> entity = new HttpEntity<>(headers);

        ResponseEntity<Map> response = restTemplate.exchange(url, HttpMethod.GET, entity, Map.class);
        // 응답 JSON에서 allowed_ips 리스트 추출 (예시 경로에 맞게 조정 필요)
        Map data = (Map) response.getBody().get("data");
        Map innerData = (Map) data.get("custom_metadata"); // custom_metadata에 IP 리스트가 있다고 가정

        List<String> allowedIps = (List<String>) innerData.get("allowed_ips");
        return allowedIps != null ? allowedIps : Collections.emptyList();
    }
}
```

## 4. ExtensionClient — 실제 시크릿 조회용 외부 API 호출

```java
@Component
public class ExtensionClient {

    private final RestTemplate restTemplate;

    public ExtensionClient(RestTemplate restTemplate) {
        this.restTemplate = restTemplate;
    }

    public String fetchSecret(String account, String secretPath) {
        String url = String.format("https://extension.example.com/api/secrets?account=%s&secretPath=%s", account, secretPath);

        HttpHeaders headers = new HttpHeaders();
        headers.set("Authorization", "Bearer <extension_access_token>");

        HttpEntity<Void> entity = new HttpEntity<>(headers);

        ResponseEntity<Map> response = restTemplate.exchange(url, HttpMethod.GET, entity, Map.class);
        Map<String, String> body = response.getBody();

        return body != null ? body.get("secret") : null;
    }
}
```

## 5. IP 추출 유틸 (Spring Security 기반)

```java
@Component
public class IpExtractor {
    public String extract(HttpServletRequest request) {
        String xfHeader = request.getHeader("X-Forwarded-For");
        if (xfHeader != null && !xfHeader.isEmpty()) {
            return xfHeader.split(",")[0].trim();
        }
        return request.getRemoteAddr();
    }
}
```
